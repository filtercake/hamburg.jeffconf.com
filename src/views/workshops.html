Serverless Web Application on AWS (beginner)

This workshop shows you how to build a dynamic, serverless web application. You'll learn how to host static web resources with Amazon S3, how to use Amazon Cognito to manage users and authentication, and how to build a RESTful API for backend processing using Amazon API Gateway, AWS Lambda and Amazon DynamoDB.

Serverless Chatbot Development (all)

How many buzzwords can you fit into a single workshop while still keeping it useful? Let’s see! This workshop will teach you how to build serverless chatbot for multiple platforms with NLP integration. It will also show you how to test your chatbot.

Topics:

  • Why serverless and chatbots are good fit
  • How to create a chatbot for fb messenger and telegram
  • How to store state in serverless chatbot
  • How to integrate you chatbot with the third-party service
  • How to send scheduled messages
  • How to integrate NLP to your chatbot
  • How to test your chatbot
  • Basics of good chatbot design

Designing, building, and operating FaaS-based microservices on Azure

In this workshop we'll explore how to build and run a FaaS-based microservices architecture on Azure. Topics include:

  • Using Domain Driven Design (DDD) to design a microservices architecture.
  • Choosing the right Azure technologies for serverless functions compute, storage, messaging, and other elements of the design.
  • Understanding microservices design patterns.
  • Designing for resiliency, scalability, and performance.
  • Building a CI/CD pipeline.

Serverless Data Processing on AWS (advanced)

This workshop demonstrates how to collect, store, and process data with a serverless application. In this workshop you'll learn how to automatically process files on Amazon S3 using AWS Lambda, how to build real-time streaming applications using Amazon Kinesis Streams and Amazon Kinesis Analytics, how to archive data streams using Amazon Kinesis Firehose and Amazon S3, and how to run ad-hoc queries on those files using Amazon Athena.

Developing Event-driven and Serverless Applications with IBM Cloud Functions & Apache OpenWhisk

Learn the basics and strengths of IBM Cloud Functions & Apache OpenWhisk. In this workshop you will learn how to develop serverless applications composed of loosely coupled microservice-alike functions. You’ll learn about the key concepts and work with our CLI and UI to become a FaaS star by implementing a real-world application. You will also learn how to make use of related technologies such as our API Gateway, Message Hub and App Connect integration as well as other event providers. Finally, you will learn how to package and deploy your entire serverless application together using the the open Serverless Framework as well as IBM’s wskdeply.
